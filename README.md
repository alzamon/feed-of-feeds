# FoF - Feed of Feeds

A hierarchical feed reader with weighted sampling based on user preferences.

## Overview

FoF (Feed of Feeds) is an advanced RSS/Atom feed reader that allows you to:

- Sample articles randomly from your feeds weighted by preference scores
- Create union feeds that combine multiple sources with custom weighting
- Apply filters using regex on title, content, author, and links
- Build hierarchical feed structures for advanced content curation

## Features

- **Weighted Sampling**: Articles are presented based on your preference history
- **Hierarchical Feeds**: Union feeds can contain regular feeds, filter feeds, or other union feeds
- **Custom Scoring**: Rate articles from 0-100 to train the recommendation system
- **Advanced Filtering**: Use regex patterns to include or exclude content
